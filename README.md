# SHADOW TIG
## Description

Ce projet permet d'acheter automatiquement **une** TIG dans le shop de l'intra 42. Il automatise le processus d'achat en bloquant les contrôles de la souris.

## Utilisation
	bash <(curl https://chahalor.github.io/tig/)
